# 项目运行环境配置清单（含下载链接）

> 适用项目：电厂巡检机器人无线视频传输系统
> 目标：保证项目可以正常启动与运行

---

## 一、摄像头端（camera-client）
用于采集摄像头画面并推流到 MediaMTX

### 1. Python 运行环境
- **Python 3.8+**
- 下载：https://www.python.org/downloads/

### 2. Python 依赖库
- 主要库：`aiortc` `opencv-python` `aiohttp` `numpy`
- 安装方式：
  ```bash
  pip install -r requirements.txt
  ```

### 3. FFmpeg（备用 RTSP 推流）
- 下载：https://ffmpeg.org/download.html
- Windows推荐：https://www.gyan.dev/ffmpeg/builds/

---

## 二、流媒体服务器（MediaMTX）
用于 WHIP 推流入口 + WHEP 拉流出口 + 录像

### 1. MediaMTX 可执行文件
- 下载：https://github.com/bluenviron/mediamtx/releases

### 2. 配置文件
- 项目内已提供：`mediamtx.yml`

---

## 三、后端服务（server）
用于用户认证与 API 接口

### 1. Python 运行环境
- **Python 3.8+**（同上）

### 2. 依赖库
- 主要库：`fastapi` `uvicorn` `sqlalchemy` `bcrypt` `python-jose`
- 安装方式：
  ```bash
  pip install -r requirements.txt
  ```

---

## 四、前端（web-client）
用于浏览器播放 WebRTC 视频

### 1. Node.js
- **Node.js 16+**
- 下载：https://nodejs.org/

### 2. 包管理器
- 推荐 `pnpm`（也可用 npm）
- pnpm 安装：https://pnpm.io/installation

### 3. 前端依赖
```bash
pnpm install
```

---

## 五、数据库
- 默认使用 **SQLite**（无需安装）
- 如需 MySQL：
  下载：https://dev.mysql.com/downloads/

---

## 六、浏览器环境
- 推荐 **Chrome / Edge 最新版**（原生支持 WebRTC）

---

## 七、设备与网络要求
- 有可用摄像头（本地或USB）
- 网络端口可用：
  - **8889**（WebRTC：WHIP/WHEP）
  - **8554**（RTSP 备用）

---
