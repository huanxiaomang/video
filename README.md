# 电厂巡检机器人端到端无线视频传输系统

## 项目简介

本项目是一个完整的端到端无线视频传输系统，专为电厂巡检机器人设计。系统通过机器人搭载的摄像头采集实时视频，经过编码压缩后传输到远程监控端，实现人工故障识别，并预留AI辅助识别接口。

## 技术指标

- ✅ 视频分辨率：≥720P (1280x720)
- ✅ 实时性要求：延迟 ≤1s
- ✅ 自适应码率：带宽受限时自动调节
- ✅ 开发语言：Python (后端) + Vue3 + TypeScript (前端)
- ✅ 传输协议：WebRTC / RTSP / HLS

## 系统架构

系统由三个主要部分组成：

### 1. 摄像头采集端 (Python)
- 视频采集 (OpenCV)
- H.264编码 (FFmpeg)
- RTSP推流
- 设备注册与心跳
- 网络质量监控

### 2. 传输服务器 (Python + FastAPI)
- RTSP/WebRTC流媒体服务
- 流转发与分发
- 码率自适应
- 录像存储
- 设备管理API
- WebSocket实时通信

### 3. 监控客户端 (Vue3 + TypeScript)
- 实时监控页（1/4/9/16分屏）
- 设备管理页
- 录像回放页
- AI检测页
- 系统设置页

## 核心功能

### 实时监控
- 多设备同时监控（支持1/4/9/16分屏）
- WebRTC低延迟播放（≤1s）
- 视频质量自适应切换
- 全屏播放、截图、录制
- 故障区域框选标注

### 设备管理
- 设备注册/注销
- 设备状态实时监控
- 设备分组管理
- 设备搜索与过滤

### 录像回放
- 时间轴选择
- 录像列表展示
- 倍速播放（0.5x/1x/2x/4x）
- 录像下载
- 关键帧截图

### AI接口预留
- 视频帧抽取接口
- 异常检测回调
- 故障识别结果存储
- 实时告警通知

## 技术栈

### 后端
- **Web框架**: FastAPI
- **流媒体**: MediaMTX / aiortc
- **数据库**: SQLAlchemy + SQLite/MySQL
- **缓存**: Redis
- **视频处理**: FFmpeg, OpenCV

### 前端
- **框架**: Vue 3 + TypeScript
- **UI组件**: Element Plus / Ant Design Vue
- **状态管理**: Pinia
- **视频播放**: Video.js / WebRTC
- **构建工具**: Vite

## 快速开始

### 环境要求
- Python 3.9+
- Node.js 18+
- FFmpeg 4.4+
- Redis 6.0+
- MySQL 8.0+ / SQLite 3.35+

### 安装步骤

```bash
# 1. 克隆项目
git clone <repository-url>
cd video-transmission-system

# 2. 安装后端依赖
cd server
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt

# 3. 安装前端依赖
cd ../web-client
npm install

# 4. 启动服务
# 启动后端
cd ../server
python src/main.py

# 启动前端
cd ../web-client
npm run dev
```
